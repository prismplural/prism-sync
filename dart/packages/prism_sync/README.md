# prism_sync

Dart bindings for the [prism-sync](../../) encrypted CRDT sync library. Generated by [flutter_rust_bridge](https://pub.dev/packages/flutter_rust_bridge) from the Rust FFI layer.

## What This Package Provides

- `PrismSyncHandle` -- opaque handle to a configured Rust sync instance
- Key lifecycle: `initialize`, `unlock`, `lock`, `generateSecretKey`, `changePassword`
- Mutation recording: `recordCreate`, `recordUpdate`, `recordDelete`
- Sync control: `syncNow`, `onResume`, `setAutoSync`, `status`, `pollEvent`
- Pairing: `createSyncGroup`, `joinFromQr`, `joinFromUrl`, `joinFromResponseJson`
- Device management: `listDevices`, `revokeDevice`
- `MemorySecureStore` for testing

## Quick Start

```dart
import 'package:prism_sync/generated/api.dart';

// 1. Create a sync instance
final handle = await createPrismSync(
  relayUrl: 'https://relay.example.com',
  dbPath: '/path/to/sync.db',  // or ":memory:" for testing
  allowInsecure: false,
  schemaJson: '''
  {
    "entities": {
      "tasks": {
        "fields": {
          "title": "String",
          "done": "Bool",
          "priority": "Int"
        }
      }
    }
  }
  ''',
  useMemoryStore: true,  // false in production (use prism_sync_flutter)
);

// 2. Initialize (first-time) or unlock (subsequent launches)
final secretKey = await generateSecretKey();  // BIP39 mnemonic
await initialize(
  handle: handle,
  password: 'user_password',
  secretKey: secretKeyBytes,  // mnemonic converted to bytes
);

// 3. Record mutations
await recordCreate(
  handle: handle,
  table: 'tasks',
  entityId: 'task-uuid-1',
  fieldsJson: '{"title": "Buy milk", "done": false, "priority": 1}',
);

// 4. Trigger sync
final resultJson = await syncNow(handle: handle);
// Returns: {"pulled": 0, "merged": 0, "pushed": 1, "duration_ms": 42, "error": null}

// 5. Check status
final statusJson = await status(handle: handle);
// Returns: {"syncing": true, "last_sync": "2026-03-15T10:30:00Z", "pending_ops": 0}

// 6. Poll for events
final event = await pollEvent(handle: handle);
// Returns JSON string or null if no events pending
```

## Pairing (Multi-Device Setup)

```dart
// Device A: Create sync group
final inviteJson = await createSyncGroup(
  handle: handle,
  password: 'shared_password',
  relayUrl: 'https://relay.example.com',
);
// inviteJson contains: qr_payload, words, url, sync_id, relay_url

// Device B: Join from QR code
await joinFromQr(
  handle: handle,
  qrBytes: qrPayloadBytes,
  password: 'shared_password',
);

// Or join from deep link URL
await joinFromUrl(
  handle: handle,
  url: 'prism://join?...',
  password: 'shared_password',
);
```

## Generated Bindings

All files in `lib/generated/` are auto-generated by flutter_rust_bridge. **Do not edit them manually.** They are regenerated whenever the Rust FFI API changes:

```bash
flutter_rust_bridge_codegen generate
```

## Platform Setup

This package requires the native Rust library to be compiled for your target platform. The compilation is handled by:

- **iOS/macOS:** Xcode build phase that compiles the Rust `cdylib`/`staticlib`
- **Android:** Gradle task that invokes `cargo build` for each ABI
- **Web:** Not currently supported (requires wasm target)

The `prism_sync_flutter` package provides the platform-specific setup and a real `SecureStore` backed by platform Keychain/Keystore. Use `MemorySecureStore` only for testing.
